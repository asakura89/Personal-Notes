Yes, Sitecore EXM (Email Experience Manager) exposes a Client API that can be used to manage subscriptions to lists in the List Manager and send automated email messages. The Email Experience Manager Client API is available in different versions of Sitecore EXM, such as version 9.0 and version 3.4.
