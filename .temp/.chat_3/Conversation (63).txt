## Bing
Entity Framework Core (EF Core) is a lightweight, extensible, open source and cross-platform version of the popular Entity Framework data access technology. EF Core can serve as an object-relational mapper (O/RM), which enables .NET developers to work with a database using .NET objects.
EF Core 7 (EF7) is the latest version of EF Core, released in November 2022. EF7 targets .NET 6 and .NET 7, and contains many features that help in porting “classic” EF6 applications to use EF7. Some of the major enhancements in EF7 include:
- JSON columns: EF7 supports mapping of aggregates built from .NET types to JSON documents stored in relational database columns. This allows querying and updating the JSON documents using LINQ.
- ExecuteUpdate and ExecuteDelete: EF7 introduces new methods that allow bulk updates and deletes without loading entities into memory.
- Faster SaveChanges: EF7 optimizes the performance of SaveChanges by reducing the number of database roundtrips and improving concurrency handling.
- Spatial types: EF7 supports mapping of spatial types such as Point, LineString, Polygon, etc. to database columns. This allows querying and manipulating spatial data using LINQ.
- Table-per-concrete-type (TPC) mapping: EF7 supports mapping inheritance hierarchies to separate tables for each concrete type in the hierarchy. This allows more flexibility in modeling complex domains.
- Migrations bundles: EF7 introduces a new way to package migrations into a single executable file that can be deployed and applied to a database without requiring the application or EF Core.
These are just some of the new features in EF7. You can find more details and examples in the What's New in EF Core 7.0 article.
