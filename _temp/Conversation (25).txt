Yes, there are open source software that utilize HTML5 storage. HTML5 introduces two mechanisms for storing structured data on the client side: local storage and session storage. Local storage is similar to cookies but with a much larger capacity and no information is sent back to the server. Session storage is similar to local storage but the data is cleared when the session ends. Both mechanisms are supported by the latest versions of pretty much every browser including Internet Explorer.
Here are some open source software that utilize HTML5 storage:
 - H5P - an open-source platform for creating and sharing rich HTML5 content and applications.
 - TrueNAS - an open-source software that provides unified file, block, and object storage.
 - OwnCloud - an open-source cloud storage platform that allows you to store files, contacts, calendars, and more.
